This source code requires 2 terminals for hosting backend and frontend each.
Using cluster database from mongodb cloud (MongoDB atlas).

To run backend on localhost:3000
{
    terminal: 
        npm install
        npm start
}

To run frontend on localhost:8080
{
    terminal:
        cd preem_pb
        npm install
        npm run serve
}

visit localhost:8080 for content of website.
localhost:3000 is for APIs.
